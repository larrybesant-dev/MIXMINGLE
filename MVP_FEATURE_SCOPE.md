- ✅ Follow/unfollow users

### Notifications
- ✅ Push notifications (Firebase Cloud Messaging)

## ❌ Out of Scope for MVP

The following features will NOT be in the initial release:

- ❌ Augmented Reality (AR) effects
- ❌ AI-powered recommendations (Phase 2)
- ❌ Live streaming to external platforms
- ❌ Advanced e-commerce
- ❌ NFT/blockchain integration
- ❌ VR support
- ❌ Desktop apps (native macOS/Windows)
- ❌ Game/interactive features
- ❌ AI content moderation (basic only)
- ❌ Scheduled streaming
- ❌ Multiple simultaneous rooms

---

## Known Limitations (MVP)

### Technical
- Maximum 50 concurrent users per room
- 30-day message history retention
- 5GB storage per user
- 100 rooms limit per user
- No offline message queuing

### Feature Limitations
- Basic AI moderation only
- Manual admin review required for reports
- No scheduled events (for MVP)
- Limited room customization
