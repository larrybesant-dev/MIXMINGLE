# MixVy Onboarding Guide

Welcome to MixVy! Follow these steps to get started:

## 1. Install Dependencies
Run:
```
flutter pub get
```

## 2. Launch the App
Run:
```
flutter run
```

## 3. Run Tests
Run:
```
flutter test
```

## 4. Project Structure
- `lib/features/` — Feature modules
- `lib/models/` — Data models
- `lib/services/` — Business logic
- `lib/router/` — Navigation
- `test/` — Unit tests

## 5. Adding Features
- Create new feature folders in `lib/features/`
- Add controllers, screens, and services
- Write tests in `test/`

## 6. Contributing
- Document new flows in README.md
- Add onboarding steps here as needed

## 7. Support
For help, contact the project maintainer or open an issue.
